# Contributing to GBM Neoantigen Discovery Pipeline

Thank you for your interest in contributing! 🎉

## Ways to Contribute

### 🐛 Reporting Bugs

If you find a bug:
1. Check [existing issues](https://github.com/zz153/GBM_Neoantigen_Pipeline/issues)
2. If not found, create a new issue with:
   - Clear title
   - Steps to reproduce
   - Expected vs actual behavior
   - R version and sessionInfo()
   - Error messages (if any)

### 💡 Suggesting Enhancements

Have an idea? Great! Create an issue with:
- Clear description of the enhancement
- Use case / motivation
- Example implementation (if applicable)

### 📝 Improving Documentation

Documentation improvements are always welcome:
- Fix typos or unclear explanations
- Add examples
- Improve code comments
- Translate documentation

### 🔧 Code Contributions

#### Setup Development Environment
```bash
# Fork and clone
git clone https://github.com/YOUR_USERNAME/GBM_Neoantigen_Pipeline.git
cd GBM_Neoantigen_Pipeline

# Create a branch
git checkout -b feature/your-feature-name

# Install dependencies
Rscript environment/requirements.R
```

#### Coding Standards

- Use meaningful variable names
- Comment complex logic
- Follow existing code style
- Add documentation for new functions
- Test your changes

#### Pull Request Process

1. Update README.md if needed
2. Add comments to your code
3. Test thoroughly
4. Commit with clear messages:
   - `feat:` New feature
   - `fix:` Bug fix
   - `docs:` Documentation
   - `refactor:` Code refactoring
   - `test:` Tests
5. Push to your fork
6. Create Pull Request with:
   - Clear title
   - Description of changes
   - Related issue number

### 📊 Adding New Features
