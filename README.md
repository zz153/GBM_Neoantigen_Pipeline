# Computational Identification of Clonal Neoantigens in IDH-Wildtype Glioblastoma

[![Research Square](https://img.shields.io/badge/Research%20Square-preprint-blue)](https://doi.org/10.21203/rs.3.rs-8351369/v1)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![R](https://img.shields.io/badge/R-%3E%3D4.3.0-blue)](https://www.r-project.org/)

> **A data-driven computational pipeline for identifying evolutionarily stable neoantigen targets in glioblastoma through integrated genomic filtering and HLA prediction**

---

## 📖 Overview

This repository contains the complete computational pipeline and analysis code for our manuscript:

**"Computational Identification of Clonal Neoantigens in IDH-Wildtype Glioblastoma Through Integrated Genomic Filtering and HLA Prediction"**

### Key Features

- 🧬 **Focused Gene Panel Approach**: Targets TP53, PTEN, and EGFR mutations
- 🎯 **Clonality-Based Filtering**: Prioritizes VAF > 0.4 mutations
- 💻 **Integrated HLA Prediction**: NetMHCpan-4.2 (Class I) + NetMHCIIpan-4.3 (Class II)
- 📊 **Comprehensive Analysis**: From TCGA data to publication-ready figures
- 🔄 **Fully Reproducible**: Modular R scripts with clear dependencies

### Results Summary

- **186 unique neoantigen peptides** (150 Class I, 36 Class II)
- **468 HLA-peptide binding predictions** across 17 alleles
- **68.5% patient coverage** (87/127 IDH-WT GBM patients)
- **98% private neoantigens** (minimal inter-patient sharing)

## 📋 Pipeline Overview

### Workflow

```
TCGA-GBM (n=617)
    ↓
[1] IDH-Wildtype Filter (n=320)
    ↓
[2] Mutation + Expression Data (n=127)
    ↓
[3] Mutation Calling (19,030 mutations)
    ↓
[4] Clonality Filter (VAF > 0.4 → 181 clonal mutations)
    ↓
[5] Expression Validation (TPM > 10 → 3 genes: TP53, PTEN, EGFR)
    ↓
[6] Neoantigen Prediction (NetMHCpan → 468 predictions)
    ↓
[7] Strong HLA Binders (IC50 < 500 nM, %Rank < 2% → 186 neoantigens)
```

### Script Descriptions

| Script | Purpose | Key Outputs |
|--------|---------|-------------|
| `01_download_data.R` | Download TCGA-GBM mutation & expression data | MAF files, RNA-seq TPM |
| `02_filter_mutations.R` | Filter IDH-WT, calculate VAF, identify top genes | Filtered MAF, Top50 genes |
| `03_expression_validation.R` | Compare tumor vs normal expression | Expression summary, Figure 1 |
| `04_clonality_analysis.R` | Identify clonal mutations (VAF > 0.4) | Clonal mutations, VAF plots |
| `05_neoantigen_prediction.R` | Run NetMHCpan predictions (Class I & II) | Strong binders, 186 peptides |
| `06_survival_analysis.R` | Kaplan-Meier by mutation status | Survival curves, Cox regression |
| `07_sharing_analysis.R` | Calculate neoantigen sharing across patients | Sharing statistics |
| `08_generate_figures.R` | Generate all publication figures | Figures 1-4, Supplementary |

## 🔧 Detailed Installation

### Step 1: R and R Packages

```r
# Install Bioconductor packages
if (!requireNamespace("BiocManager", quietly = TRUE))
    install.packages("BiocManager")

BiocManager::install(c(
  "TCGAbiolinks",
  "SummarizedExperiment",
  "recount3",
  "edgeR"
))

# Install CRAN packages
install.packages(c(
  "survival",
  "survminer",
  "ggplot2",
  "dplyr",
  "tidyr",
  "cowplot",
  "gridExtra",
  "httr",
  "jsonlite"
))
```

### Step 2: NetMHCpan Tools (Required)

**NetMHCpan-4.2** (HLA Class I):
1. Request access: https://services.healthtech.dtu.dk/services/NetMHCpan-4.2/
2. Download and install
3. Update path in `scripts/05_neoantigen_prediction.R` (line 218)

**NetMHCIIpan-4.3** (HLA Class II):
1. Request access: https://services.healthtech.dtu.dk/services/NetMHCIIpan-4.3/
2. Download and install
3. Update path in `scripts/05_neoantigen_prediction.R` (line 290)

> **Note**: Academic license required. Approval typically takes 1-2 business days.

## 📊 Key Results

### Neoantigen Discovery

- **Total Predictions**: 468 HLA-peptide interactions
  - Class I (9-mers): 402 predictions (85.9%)
  - Class II (15-mers): 66 predictions (14.1%)

- **Strong Binders**: 186 unique peptides
  - TP53: 79 peptides
  - PTEN: 76 peptides
  - EGFR: 31 peptides

### Patient Coverage

- **87 of 127 patients** (68.5%) have at least one strong binder
- **Class I only**: 68 patients (53.5%)
- **Both Class I & II**: 19 patients (15.0%)
- **No predictions**: 40 patients (31.5%)

### Clinical Implications

- **Survival**: TP53 mutations associated with worse OS (p=0.028)
- **Sharing**: 98% neoantigens are patient-specific
- **Efficiency**: 77% faster than WES (3 weeks vs 13 weeks)

## 🐛 Troubleshooting

### Common Issues

**1. NetMHCpan not found**
```bash
# Update paths in scripts/05_neoantigen_prediction.R
netmhcpan_path <- "/your/path/to/netMHCpan-4.2/netMHCpan"
```

**2. TCGA download timeout**
```r
# Increase timeout in scripts/01_download_data.R
options(timeout = 600)  # 10 minutes
```

**3. Memory issues**
```r
# Increase R memory limit
memory.limit(size=16000)  # 16GB (Windows)
```

**4. Missing GTEx data**
```bash
# Pipeline continues without GTEx
# Uses TCGA normal samples only
```

See [docs/FAQ.md](docs/FAQ.md) for more solutions.
